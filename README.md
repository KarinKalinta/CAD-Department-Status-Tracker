# Project 011 - CAD Status Tracker

## Purpose

This project contains the working materials for the CAD department status tracker, including Excel workbook exports, Python/PowerShell helper scripts, and the Office Script prototype.

The live production workbook is stored outside this Codex workspace:

`C:\Users\User\Desktop\All\CAD Status\CAD_Status_Tracker_Batch_Aware_MANAGER_VIEW_LIVE_FORMULAS.xlsm`

## Folder Structure

- `source_files/` - Reserved for source workbook inputs or raw files.
- `scripts/` - Python, PowerShell, and Office Script files used to create or repair tracker workbooks.
- `assets/` - Reserved for images or visual assets.
- `references/` - Reserved for reference material.
- `exports/` - Generated Excel workbook versions and tracker drafts.
- `notes/` - Planning notes and organization documents.
- `archive/` - Reserved for unclear or old files when they need to be separated.
